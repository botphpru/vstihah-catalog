# Веб-сайт с каталогом стихов и поздравлений «В стихах» (vstihah.ru)

## Описание проекта
Данный проект представляет собой легковесную, независимую от тяжелых сторонних фреймворков и SEO-оптимизированную контентную платформу-каталог. Сайт предназначен для точечного подбора поэтических произведений и поздравлений с помощью развитой системы фасетной фильтрации.

## Системные требования и окружение
Для запуска и проверки проекта необходимы:
- **Веб-сервер:** Apache 2.4 с активированным модулем `mod_rewrite` (настройки перенаправления находятся в файле `.htaccess`).
- **Интерпретатор:** PHP версии 8.1 (или выше).
- **СУБД:** MySQL версии 8.0 (или выше).
- **Служба кэширования:** Memcached (для оптимизации SQL-выборок).
- **Сборщик зависимостей:** Composer (для автозагрузки классов по стандарту PSR-4).

## Инструкция по сборке и запуску
1. Склонируйте или скачайте репозиторий в рабочую директорию веб-сервера (например, `www/vstihah.ru/`).
2. Импортируйте дамп базы данных (файл `database.sql` из материалов проекта) в вашу СУБД MySQL.
3. В корне проекта выполните команду `composer install` (или `composer update`) для генерации файлов автозагрузки в папке `vendor/`.
4. Настройте параметры подключения к базе данных в конфигурационном файле `config/init.php` (используйте шаблон из `config/init.example.php`).
5. Перенаправьте корневую директорию домена (Document Root) на публичную папку `public/`. Единая точка входа — `public/index.php`.

## Кейсы тестирования (проверка работоспособности)
1. **Тест маршрутизации и ЧПУ:** Переход по адресу `/` открывает главную страницу. Переход по каноническому ЧПУ-пути (например, `/свадьба/дочери/`) открывает отфильтрованный каталог. Переход по несуществующему адресу вызывает контролируемое исключение `NotFoundException` и отдает страницу ошибки 404.
2. **Тест каноничности (301 редирект):** При попытке зайти по неканоническому адресу (нарушающему порядок справочников) система автоматически выполняет 301-редирект на правильный канонический URL.
3. **Тест интерактивного рейтинга (AJAX):** При клике на кнопки голосования за стихотворение фронтенд отправляет асинхронный POST-запрос с CSRF-токеном. Счетчик в DOM-дереве изменяется мгновенно без перезагрузки страницы.
